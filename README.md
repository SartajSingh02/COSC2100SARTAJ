# Tic Tac Toe Game in C# (WPF)

## Project Information
- **Name**: Sartaj Singh
- **Date**: 2024-10-08
- **Modified**: 2024-10-08
- **Description**: 
  - This C# WPF Tic Tac Toe game allows two players to take turns playing "X" and "O". The game checks for win conditions (rows, columns, and diagonals), handles draws, and updates player scores. The game features options to reset the board, reset scores, and exit the application.

## Features
- **Two-player game**: Allows two players to enter their names and take turns.
- **Win condition check**: Automatically checks for a winner after every move.
- **Draw detection**: Detects and announces a draw when the board is full.
- **Score tracking**: Tracks and updates the score for both players as well as the number of draws.
- **Reset and Exit**: The game can be reset, and scores can also be reset.
- **Customized design**: Button colors and win highlighting are personalized with `#FFB9B7B7`.

## Installation
1. Clone or download the repository.
2. Open the solution file in Visual Studio.
3. Build and run the project.

## How to Play
1. Enter the names of the two players in the respective fields.
2. Press the "Choose Starting Player" button or start with Player X.
3. Take turns clicking on the buttons to place "X" or "O".
4. The game will automatically check for a winner or announce a draw.
5. Press the "Reset" button to play again, or "Exit" to close the game.
